Shipping Logistics


What it is

A shipping company's internal system for tracking each of its transaction/contract.


Technologies

Front end: HTML, CSS, Bootstrap, Javascript
Back end: Django, MongoDB


Some Feature Highlights

  • Filtering through contracts
  • Adding, editing, removing new contracts and business contacts
  • Add updates to a contract regarding its progress
  • Generate a report for a contract
  • Able to upload a signature (photo) when a transaction is completed
  • Customized Django admin panel
  • etc
