FeastFreedom Showcase


What it is

A peer-to-peer food service platform. A user can register an account to be a food provider. And other users will be able to make order from them.


Technologies

Front end: React, Redux, Shards-UI
Back end: Django, Django REST Framework, PostgreSQL


Some Feature Highlights

  • React in-form field check to reduce traffic to the backend API
  • Dynamic rendering of UI elements without the need to reload pages
  • State management with Redux on the frontend
  • Custom user model on the backend
  • JWT authentication
  • etc
