Mixtaped Showcase


What it is

Mixtaped is the spotify for local and non-big-name musicians. Cofounded by Jacob Levitt, Michal Cymbalisty, and me in 2017.


Technologies

Front end: HTML, CSS, Bootstrap, Javascript
Back end: Nodejs, Express, MongoDB


Features I implemented

user features, upload songs/videos, search, music playing capability, reacting on a song - including liking and commenting.
* the web application is built entirely by myself.
